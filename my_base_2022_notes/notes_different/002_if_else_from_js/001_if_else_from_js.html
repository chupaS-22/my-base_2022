IF ELSE в JAVASCRIPT. Условный оператор "?". Примеры IF ELSE.

Что если выполнение того или иного кода требуются только при определенном условии тут нам на помощь приходит инструкция if a помогут ей условный оператор ? и блок else.

Разберем сразу на примере.

У нас есть переменная let message = "Привет!!!"

И мы хотим выводить в консоль эту переменную, только если выполниться некое условие, для этого мы пишем инструкцию.

Условие внутри фигурных{...}скобок выполниться, только в том случае, когда условие внутри крулых(...) скобок вернетtrue, то есть истина. В данном случае2 > 1. Это истина.true. Поэтому код выполниться, и мы увидим в консоли строку Привет!!!Код сработает.

Если же, поменять условие поменять, к примеру на:2 > 5то оно конечно же не выполниться, так как вернетсяfalse;2 никак не больше 5, а это значит, что в консоли ничего не отобразиться, строка останется пустой, другими словами - код не выполниться.

Необязательный блок "else"

"Необязательный" блок elseдает возможность выполнить вторую часть кода, при условии если первый код не выполнен, и вернул false.

для того чтобы проверить несколько условий, мы можем использовать блок "else if "

Блоки "else"и "else if "- являются необязательными и используются, только при необходимости.

Условный оператор "?"

Чтобы понять как он работает, разберем пример.

С помощью условного оператора "?", мы можем переписать это классическое условное ветвление.

Причем скобки let = messageEnd = (5 > 1)из условия мы можем убрать, так как условный оператор "?"имеет очень низкий приоритет, и сначала выполняться условие, а уже потом условный оператор

Следует добавить, что условный оператор "?"является тернарным, это единственный оператор у которого, целых три операнда.

С помощью условного оператора "?"можно также реализовывать более сложную конструкцию включающие в себя блоки elseи else if.

Такая реализация менее читабельна чем классическая, поэтому советуют использовать все таки классическую реализацию, по этой же причине не стоит использовать условный оператор как полный заменитель if else.
